# Jig

An general purpose data processing engine for the BEAM, written in Elixir.

## Installation

Add `jig` to your list of dependencies in `mix.exs`

```elixir
def deps do
  [
    {:jig, "~> 0.0.1"}
  ]
end
```

## Help Wanted :)

If you think this `jig` thing might be worthwhile and you don't see a feature
